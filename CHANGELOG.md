# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [Unreleased]

### Added

- Apache-2.0 licensing and repository governance documents for public open-source release.
- Contributing, CLA, security, release, notice, and code-of-conduct policies.
- GitHub issue forms, PR template, CODEOWNERS, Dependabot, CLA workflow, audit workflow, release workflow, stale triage, and repository tooling files.

### Changed

- Repository metadata and README now align with the public licensing and contribution model.

## [0.1.0] - 2026-03-14

### Added

- Initial `xeno-lib` release covering image transforms, media utilities, AI-assisted processing, and the `xeno-edit` CLI.
- Feature-gated video encode/decode, audio encode/decode, subtitle handling, text overlay, document processing, and SVG vectorization.
- FFmpeg parity tracking and competitor benchmark infrastructure.
